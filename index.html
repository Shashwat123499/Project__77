Project on Console

Visual Studio code is a code editor used to create applications of different languages, such as c++, html, etc. To set it up, first download it from microsoft's official website or the microsoft store (windows 8 and up) and then set it up. Then, create a folder, open it with visual studio code and start working on your file.



Console screen is used for debugging purposes and to check if the output of the code is correct or not, before the code is displayed on the actual website screen.
